# JavaScript Assignments

This folder contains JavaScript assignments for improving problem-solving skills and core JavaScript concepts.

---

## 📁 Assignment Categories

### Easy
Basic problems to build logic.

- **Anagram** → Check whether two strings contain same characters.
- **Expenditure Analysis** → Find category-wise total spending.
- **Find Largest Element** → Return largest number from array.

---

### Medium
Intermediate level problems.

- **Count Vowels** → Count total vowels in a string.
- **Palindrome** → Check if string is same forward/backward.
- **Times** → Practice time/date logic.

---

### Hard
Advanced logic-based assignments.

- **Calculator** → Build calculator functions.
- **To-Do List** → Create task manager logic.
